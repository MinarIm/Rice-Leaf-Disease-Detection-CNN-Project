# 🌾 Rice Leaf Disease Detection using CNN

A Deep Learning project for Digital Image Processing course.

## 📌 Project Overview
Detects 3 types of rice leaf diseases using a Convolutional Neural Network (CNN):
- Bacterial Leaf Blight
- Brown Spot
- Leaf Smut

## 📁 Dataset
[Rice Leaf Diseases Dataset - Kaggle](https://www.kaggle.com/datasets/vbookshelf/rice-leaf-diseases)
- Total Images: 120 (40 per class)

## 🧠 Model Architecture
- 4 Convolutional Blocks with BatchNormalization
- MaxPooling layers
- Dropout for regularization
- Softmax output for 3 classes

## 🛠️ Technologies Used
- Python
- TensorFlow / Keras
- Google Colab
- NumPy, Matplotlib, Seaborn
- Scikit-learn
